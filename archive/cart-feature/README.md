# Retired Cart Feature

This folder preserves the complete browser-only combined-request cart that was removed from the published SHIPS website on June 13, 2026.

The live website now uses each garment's existing primary action only:

- Shopify checkout links continue to show `Buy Now`.
- Request-only garments continue to open the existing email request.
- Coming-soon and sold-out garments remain disabled.

The archived files are not loaded by the published website. To restore the cart later, use these files as a reference and deliberately re-integrate them with the then-current public site and editor.
